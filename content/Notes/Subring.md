#### Definition
Given a [[ring]] $(R, +, \cdot)$, $\emptyset \neq S \subseteq R$ is a subring if $(S, +, \cdot)$ is a ring with the induced operations.

Equivalently, $(S, +, \cdot)$ is a subring of $(R, +, \cdot)$ if and only if $(S, +) \leq (  R, + )$ and $S$ is stable under products.

Denote $S \leq (R, +, \cdot)$ if $S$ is a subring.

If $R$ is unital, and $S \leq R$, $1_{R} \in S$, we say $S$ is a unital subring (note that this is note necessary of all subrings of unital rings).
