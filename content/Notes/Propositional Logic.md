#### The Language of Propositional Logic

Consists of:
- Variables: $v_{0},v_{1},v_{2},\ldots, v_{n}, \ldots$ for $n\in \mathbb{N}$.
- Logical symbols: $\neg, \wedge, \vee, \rightarrow, \leftrightarrow$.
- Parentheses: $(,)$.

#### Propositional Formulas
The set of propositional formulas is defined as follows:
1) Each proposition variable is a propositional formula.
2) If $\varphi$ is a propositional formula, then so is $\neg \varphi$.
3) If $\varphi$ and $\psi$ are propositional formulas, then so are $(\varphi \wedge \psi), (\varphi \vee \psi), (\varphi \rightarrow \psi), (\varphi \leftrightarrow \psi)$.
4) A string of symbols is a propositional formulas if and only if it is generated from 1) by finitely many applications of 2) and 3).
