#definition #abstract-algebra/vector-spaces

#### Definition
If $S,T \in \mathscr{L}(V,W)$ and $\lambda \in \mathbb{F}$, define,
1) $(S+T)(v) = Sv + Tv$.
2) $(\lambda T)(v) = \lambda (Tv)$.
Which are an addition and scalar multiplication on $\mathscr{L}(V,W)$.

#### Theorem
$\mathscr{L}(V,W)$ is a vector space over $\mathbb{F}$.
