#definition #abstract-algebra/ring-theory/polynomials 

#### Definition
For a commutative [[ring]] $R$ with unity, the set of [[Polynomial|polynomials]] $R[x] = \{ P(x) \mid P(x) \text{ polynomials} \}$ is a ring with coefficient-wise addition, and multiplication (the same as in the [[Group Algebra|group algebra]] for $G=( \mathbb{Z}, + )$) given by the following properties,
1) $x^ix^j = x^{i+j}$.
2) Follows commutativity and distributivity.

#### Properties
- $\deg(P + Q) \leq \max(\deg P, \deg Q)$ (is strict if $LT(P) = - LT(Q)$).
- If $R$ is a domain (and $P,Q \neq 0$), $\deg(PQ) = \deg P + \deg Q$.
- If $R$ is a domain, then $R[x]$ is a domain.
