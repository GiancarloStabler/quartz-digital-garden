#### Definition
Given commutative ring $(R, +, \cdot)$ with $1_{R}$, and ideal $I \unlhd R$. We have $( I, + ) \leq (  R, + )$, which implies $I$ is a normal subgroup, and therefore there exists a commutative group $( {^{\textstyle R}\big/_{\textstyle I}}, + ) = \{ r + I \mid r\in R \}$.

#### Examples
1) ${^{\textstyle R}\big/_{\textstyle (0)}} \simeq R$.
2) ${^{\textstyle R}\big/_{\textstyle (1_{R})}} = {^{\textstyle R}\big/_{\textstyle R}}= 0$ ring.
3) ${^{\textstyle \mathbb{Z}}\big/_{\textstyle n\mathbb{Z}}}\simeq \mathbb{Z}_{n}$ as a ring.
