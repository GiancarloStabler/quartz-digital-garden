#### Definition
Two matrices $A,B$ are similar if there exists some matrix $S$ such that,
$$
A = S^{-1} B S
$$
for some $S,S^{-1}$ such that $S \cdot S^{-1} = S ^{-1} \cdot S = I_{n}$.

Similar matrices have the same [[trace]], [[determinant]], and [[characteristic polynomial]]. In particular, we can define $\operatorname{tr}T$, $\det T$, or $f_{T}(x)$ by using any of these matrices.
